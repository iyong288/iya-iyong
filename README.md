Program kasir sederhana
Program ini dibuat untuk melatih penggunaan **function** dalam bahasa Python.  
Program berfungsi sebagai kasir sederhana yang dapat:
- Menampilkan daftar barang
- Menghitung total harga pembelian
- Menghitung diskon jika total mencapai syarat tertentu
- Menghitung kembalian
